Secure IoT MQTT Architecture (ESP32)

This project demonstrates 3 levels of security in IoT communication using MQTT and ESP32.

It is designed as a progressive learning architecture, starting from no security to a fully secured TLS-based system.

 Project Structure
├── 1-no-security/           → Basic MQTT (no protection)
├── 2-authentication/        → MQTT with username/password
├── 3-tls-authentication/    → MQTT over TLS + authentication (most secure)
